# Multi-Wire Cable Tester

A digital logic-based system designed to test multi-wire cables and identify faulty or disconnected connections.

## Project Overview
The system checks the integrity of individual wires within a multi-wire cable and provides visual feedback using LEDs to indicate connection status. The project demonstrates practical applications of digital logic design and circuit implementation.

## Features
- Detection of faulty or disconnected wires
- LED-based visual status indication
- Digital logic implementation
- Circuit design and testing
